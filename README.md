# SentiVibe: Next-Gen Mood, Weather-Based Music Recommender & Safety Suite

SentiVibe is an intelligent, multi-functional application designed to revolutionize how users interact with music while ensuring a safe digital environment. The system intelligently captures environmental parameters (weather) and emotional states (mood) to curate the perfect dynamic musical playlist, while simultaneously running a real-time cyberbullying detection engine to protect users from toxic online behavior.

---

## 🚀 Key Features

### 1. 🌤️ Weather-Driven Music Recommendation
* Automatically detects or predicts the user's local weather conditions (e.g., Rainy, Sunny, Gloomy).
* Dynamically maps atmospheric vibes to tailored musical frequencies (e.g., Lo-Fi beats for rainy days, energetic pop for sunny weather).

### 2. 🧠 Custom Mood Selector (Manual Control)
* Allows users to explicitly input their current emotional state (e.g., Happy, Stressed, Motivated, Relaxed).
* Instantly overrides environmental factors to generate a playlist that either elevates or balances the user's emotional wavelength.

### 3. 🛡️ Real-Time Cyberbullying Detection
* Integrates a text processing/classification module to scan incoming user interactions or chats.
* Automatically flags, filters, and alerts users about toxic phrases, hate speech, or cyberbullying patterns to maintain a healthy workspace/chat environment.

---

## 🛠️ Tech Stack & Architecture

* **Language:** Python 3.x
* **Core Framework:** Streamlit / Flask (Modify based on your frontend)
* **APIs Used:** OpenWeatherMap API (for real-time weather parameters)
* **Data Science & ML:** Pandas, NumPy, Scikit-Learn (for text classification/bullying detection)
* **Music Streaming Integration:** Spotify Web API / YouTube Embedded Player
